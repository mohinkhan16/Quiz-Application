# Quiz-Application

A responsive and interactive Quiz Application built using HTML, CSS, and JavaScript. The application displays multiple-choice questions, tracks the user's score, includes a countdown timer for each question, and provides a detailed result summary at the end of the quiz.

## Features

* Multiple-choice quiz questions
* 30-second timer for each question
* Automatic navigation to the next question
* Real-time score tracking
* Question progress indicator (e.g., Question 1 of 10)
* Restart quiz functionality
* Detailed result summary with correct and selected answers
* Responsive and modern UI design

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

1. Download or clone the repository.

```bash
git clone <repository-url>
```

2. Open the project folder.

```bash
cd Quiz-App
```

3. Open `index.html` in your browser.

No additional installation or dependencies are required.

## Application Flow

1. Quiz starts automatically.
2. User selects an answer for each question.
3. Score is calculated based on correct answers.
4. A 30-second timer runs for every question.
5. If time expires, the quiz automatically moves to the next question.
6. After all questions are completed, a result page displays:

   * Total score
   * Correct answer for each question
   * User selected answer
   * Not Attempted status (if unanswered)
